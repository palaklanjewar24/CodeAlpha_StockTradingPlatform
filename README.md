# Stock Trading Platform

## Overview

Stock Trading Platform is a Java console-based application developed as part of the CodeAlpha Java Programming Internship. It simulates a simple stock market where users can buy and sell stocks and manage their portfolio.

## Features

* View available stocks
* Buy stocks
* Sell stocks
* Manage portfolio
* Track account balance

## Technologies Used

* Java
* ArrayList
* Classes and Objects
* Loops
* Conditional Statements
* Object-Oriented Programming (OOP)

## How It Works

1. User starts with a predefined balance.
2. Available stocks are displayed.
3. User can buy stocks if sufficient balance is available.
4. User can sell owned stocks.
5. Portfolio and balance are updated automatically.
